# EasyLabel
	基于QT的标注工具
	主要有以下功能
	1.标注显示、历史标注文件读取和标注自动保存功能
	2.操作记录功能，下次打开软件会载入最后一次标注的图片
	3.标注统计功能，采用饼状图可视化展示每个类的的分布情况
	4.xml记录用户信息功能
	5.跨平台，得益于QT强大的跨平台能力，分别编译了windows和linux版本
